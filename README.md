# Data-Analyst
This is a beginner-level data analysis project created to practice and demonstrate fundamental skills in data analytics.
##  Data Analysis Projects

This repository contains two beginner-level data analysis projects created for learning and practice purposes.

---

###  Airbnb Data Analysis
This project focuses on analyzing Airbnb listings data to explore factors influencing pricing and availability.

**Key steps:**
- data cleaning and preprocessing  
- exploratory data analysis (EDA)  
- basic statistical analysis  
- visual exploration of trends and patterns  

The goal was to understand the dataset and practice analytical thinking rather than build predictive models.

---

###  Data-Related Jobs in the USA
This project analyzes data related to various data-oriented professions in the United States (e.g. Data Analyst, Data Scientist, Data Engineer).

**Key steps:**
- analysis of job roles and requirements  
- comparison of salaries and locations  
- exploration of trends in the data job market  
- basic visualizations  

The project was created to practice working with real-world, non-technical datasets and to gain insight into the data job market.
